World Cup Insights (1930–2014)

Project Overview

This dashboard analyzes all FIFA World Cup matches from 1930 to 2014. The project focuses on four core questions:

  • How did global participation expand over time?
  • How many matches were played in each World Cup?
  • How did stadium attendance change across tournaments?
  • Which countries scored the most goals overall?

Dataset

Data was sourced from publicly available historical World Cup match results. After cleaning the dataset in Excel (removing nulls, standardizing country names, fixing host country labels), the final dataset was imported into Tableau.

Key Insights

  • Participation growth: The number of participating nations expanded significantly as new regions joined.
  • Match volume increased: New tournament formats increased total matches per World Cup.
  • Attendance trends: Stadium capacity and global interest drove steady attendance growth.
  • Top scoring nations: A small group of countries consistently dominated offensive performance.

Interactive Tableau dashboard exploring World Cup participation, matches, attendance, and top-scoring nations from 1930 to 2014.
